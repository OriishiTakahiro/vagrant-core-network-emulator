# Create CORE environment with Docker

## Environment

Author's environment

- OS: Manjaro 18.0.4 Illyria
- Kernel:  x86\_64 Linux 4.19.32-1-MANJARO
- VirtualBox  (v6.0.4)
- Vagrant     (v2.2.4)
- Ansible     (v2.7.9)

## Usage

```sh
$ vagrant add 
$ debian/stretch64
$ vagrant up
$ vagrant reload
# Launch CORE on virtualbox and continue following command
$ vagrant provision
$ vagrant reload
```
